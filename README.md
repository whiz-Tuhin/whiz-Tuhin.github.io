# whiz-tuhin.github.io

Source code for Tuhin Khare's personal website, hosted on GitHub Pages.

This website is powered by [SvelteKit](https://kit.svelte.dev/) and
[Tailwind CSS](https://tailwindcss.com/). The design is inspired by
[Eric Zhang's website](https://github.com/ekzhang/ekzhang.github.io).

To develop, run:

```sh-session
$ npm install
$ npm run dev
```
